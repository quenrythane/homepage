Lekcja 03.04 Opowieść o wężu, kebabie, wielbłądzie i Pascalu

Nazywanie rzeczy- najtrudniejsze w programowaniu jest wymyślanie nazw
- bez polskich znaków
- nie używamy spacji
- najlepiej jest używać angielskich słów
- trzymamy się jednego schematu (konsekwencja jest istotna)

- camelCase- mySuperBoldHeader
każde nowe słowo zaczyna się z dużej litery i łączy się z końcem poprzedniego

- PascalCase-MySuperBoldHeader
tak jak camelCase ale pierwsza litera też jest duża

- kebab-case- my-super-bold-header
nazywamy rzeczy małymi literami i oddzielamy myślnikami(bez spacji!)

- snake_case- my_super_bold_header
tak jak przy kebabie, tylko zamiast myślnika jest podkreślenie

Można tak robić ale jest to głupie

prawdziwe selektory - #id
- .class
- element

(znacznik class=”name”> treść (/znacznik>
.name {
właściwość: wartość;
}

W tym paragrafie użyłem następującej treści:
(p class=”co-to-jest-klasa”> paragraf (/p>
.co-to-jest-klasa {
background-color: turquoise;
color: purple;
}

(znacznik id=”name”> treść (/znacznik>
.name {
właściwość: wartość;
}

W tym paragrafie użyłem następującej treści:
